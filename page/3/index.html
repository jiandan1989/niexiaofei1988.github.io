字符串：

JS字符串是由Unicode字符、数字、标点符号等组成的序列。

var str ="字符串"; or var str = '字符串'在JS中，包裹字符串单引号与双引号没有任何区别。但是还是建议使用双引号。
var str = '""'//定义了一个仅含有一个双引号的字符串。
var str = "''"//定义了一个仅含有一个单引号的字符串。
var str = " "//定义一个包含空格的字符串

js基础知识二

javascript中的原型：

在javascript中所有的函数都有prototype属性，该属性是一个对象，对象是所有属性的集合，所有的对象都有一个proto属性，所有对象的proto都指向其构造器的prototype。当函数没有任何属性，prototype对象默认有constructor（指向函数本身）属性。

function fn(){
    return '10'
    }
var fn2 = new fn();//从fn方法里面new出一个对象，生成对象
console,log(fn2.__proto__===fn.prototype)为true

所有构造器的proto都指向Function.prototype.完全继承了prototype的属性以及方法。
